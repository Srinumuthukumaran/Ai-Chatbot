**AI-Chatbot**

An AI-powered conversational chatbot application developed using Spring Boot and OpenAI APIs to generate intelligent real-time responses.
The project demonstrates full stack AI application development with seamless frontend-backend communication, REST API integration, 
and dynamic user interaction.

**Features**

* Real-time AI-generated conversational responses
* Intelligent query processing using OpenAI APIs
* Responsive and interactive chat interface
* REST API integration for frontend-backend communication
* Scalable backend architecture using Spring Boot
* Dynamic message handling and response rendering
* Clean and user-friendly UI experience
